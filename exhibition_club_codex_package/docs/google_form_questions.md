# Google Form 질문 구성

## 1. 7월 정기 관람 참석 설문

- 이름: 단답형, 필수
- 카카오톡 이름: 단답형, 선택
- 참석 가능한 일정: 객관식, 필수
  - 7.11.(토) 16:00 / 28,000원
  - 7.29.(수) 19:00 / 14,000원 / 문화의날 50% 할인
  - 둘 다 가능
  - 이번 달은 참석 어려움
- 관람 후 티타임 가능 여부: 객관식, 필수
  - 가능
  - 어려움
  - 당일 상황 보고 결정
- 티켓 비용 확인: 객관식, 필수
  - 확인했습니다
- 기타 의견: 장문형, 선택
- 개인정보 수집 및 이용 동의: 체크박스, 필수

## 2. 전시·공연 추천 설문

- 추천자 이름: 단답형, 필수
- 전시/공연명: 단답형, 필수
- 장소: 단답형, 필수
- 기간: 단답형, 선택
- 관람료: 단답형, 선택
- 추천 이유: 장문형, 필수
- 관련 링크: 단답형, 선택
- 함께 가고 싶은 날짜/시간: 단답형, 선택
- 도슨트/오디오가이드 정보: 단답형, 선택
- 주변 카페/식사 장소 또는 기타 의견: 장문형, 선택
- 개인정보 수집 및 이용 동의: 체크박스, 필수
